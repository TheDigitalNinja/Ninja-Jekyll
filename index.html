Bootstrap

Bootstrap based theme features a fully responsive design. Perfect for phones or desktops and everything inbetween.

Static Files

Without server side code static sites are very secure and with the simplest of CDN's can offer great protection against DDOS attacks.

Host Anywhere

Use aws s3 buckets, github pages, or your own apache or nginx server to host your site the way you want.

Photo Gallery

The built in gallery plugin allows you to showcase your photos in both thumbnails and full sized images.

Font Awesome

Over 675 built in icons to choose from.

Scroll Reveal

Use the scroll reveal plugin to easily add animations to any section of the website.
